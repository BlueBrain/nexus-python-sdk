Nexus Python SDK¶

A Python wrapper for the Blue Brain Nexus REST API.

Getting Started¶

Installation¶

pip install nexus-sdk

Usage¶

import nexussdk as nexus

nexus.config.set_environment(DEPLOYMENT)
nexus.config.set_token(TOKEN)

nexus.permissions.fetch()
